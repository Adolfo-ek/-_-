Manejo de Memoria Estática y Dinámica en Python
Descripción del proyecto

Este proyecto aborda el uso de memoria estática y memoria dinámica mediante implementaciones prácticas en Python. El objetivo es analizar las diferencias conceptuales y operativas entre ambos tipos de memoria, así como su impacto en el uso de recursos y la flexibilidad de los programas.

El archivo integra ambos enfoques en un solo programa para facilitar su comparación y estudio.

Objetivo

Comprender el concepto de memoria estática y memoria dinámica.

Analizar el comportamiento de cada tipo de memoria.

Aplicar estructuras de datos estáticas y dinámicas.

Comparar eficiencia, flexibilidad y uso de memoria.

Memoria estática

La memoria estática se caracteriza por reservarse en tiempo de compilación o al inicio de la ejecución del programa. Su tamaño es fijo y no puede modificarse durante la ejecución.

Características principales

Tamaño definido previamente.

No permite crecimiento ni reducción.

Acceso directo y rápido.

Menor flexibilidad.

Uso predecible de memoria.

Uso en el proyecto

Se utilizan estructuras con tamaño fijo para representar datos cuyo número de elementos no cambia durante la ejecución.

Memoria dinámica

La memoria dinámica se reserva en tiempo de ejecución y permite que las estructuras de datos cambien de tamaño según las necesidades del programa.

Características principales

Tamaño variable.

Mayor flexibilidad.

Permite inserciones y eliminaciones.

Uso de memoria adaptable.

Ligero costo adicional en gestión.

Uso en el proyecto

Se emplean estructuras dinámicas que permiten agregar, modificar y eliminar datos durante la ejecución del programa.

Tecnologías utilizadas
