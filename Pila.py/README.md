# Simulación de Pila (Stack) en Python

## Descripción

Este proyecto implementa una **simulación gráfica de una estructura de datos tipo Pila (Stack)** utilizando Python y la librería Tkinter.

Una pila sigue el principio **LIFO (Last In, First Out)**, lo que significa que el último elemento en entrar es el primero en salir.

La aplicación permite visualizar gráficamente cómo funcionan las operaciones básicas de una pila.

## Funcionalidades

* Insertar elementos en la pila
* Eliminar el elemento en la cima
* Visualizar el elemento en la cima
* Vaciar toda la pila
* Visualización gráfica de los elementos
* Indicador del tamaño actual de la pila

## Tecnologías utilizadas

* Python
* Tkinter (interfaz gráfica)

## Operaciones de la pila implementadas

| Operación   | Descripción                             |
| ----------- | --------------------------------------- |
| Insertar    | Agrega un nuevo elemento a la pila      |
| Eliminar    | Remueve el elemento de la cima          |
| Ver cima    | Muestra el elemento que está en la cima |
| Vaciar pila | Elimina todos los elementos             |

## Estructura del programa

El programa utiliza:

* Lista de Python para almacenar los elementos
* Canvas de Tkinter para la representación gráfica
* Botones para ejecutar las operaciones

## Ejecución

1. Tener Python instalado.
2. Ejecutar el archivo:

```bash
python pila.py
```

## Ejemplo de uso

1. Escribir un valor en el campo de entrada
2. Presionar **Insertar**
3. El elemento aparecerá en la pila
4. Usar **Eliminar** para quitar el último elemento insertado

## Conceptos de estructuras de datos aplicados

* Pila (Stack)
* LIFO (Last In First Out)
* Manipulación dinámica de estructuras
